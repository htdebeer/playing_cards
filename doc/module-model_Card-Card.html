Card

model/Card~ Card

A Card models a playing card.

Constructor

new Card(specification, deck, faceUpopt)

Source:
  • model/Card.js, line 177
Create a Card based on a specification in terms of suit, rank, and the deck it belongs to. Optionally, you can indicate if the card should be face up initially or not. By default a card faces down. Based on this specification the card's color is determined. Note. You cannot create a joker card using this constructor. Us the static @see joker method to create a joker card.
Parameters:
Name Type Attributes Default Description
specification Object the card's specification
Properties
Name Type Description
suit symbol the suit of the card
rank symbol the rank of the card
deck Deck the deck the card belongs to. Each card belongs to a deck.
faceUp boolean <optional>
 false is the card facing up initially or not?
